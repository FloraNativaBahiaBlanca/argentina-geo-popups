# Argentina Geo Popups

## Desarrollo local

Requisitos: Node.js y npm.

Pasos:

```sh
# Step 1: Install dependencies.
npm i

# Step 2: Start the development server.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
